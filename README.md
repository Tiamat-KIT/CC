### 会社

経営理念として「かかわった人全てをハッピーに」

- 幅広い事業概要

クライアントのWeb集客サポート
Webサイトを作ったり、Web広告、SNS運用、動画制作、サイト分析など

#### ポスティング

#### コンシェルジュ
ココカラという転職・縁結び・家づくりナビなどのサービスを行っている
